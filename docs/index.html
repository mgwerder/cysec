What is Cysec?

Cysec is a open-source, extensible and flexible web application written in Java.

Its main purpose is to provide a platform for so called "coaches". Coaches are interactive pieces of software that assess a users skill or progress on a certain topic, e.g cyber security. It is developed by the University of Applied Science FHNW in Switzerland as part of the SMESEC EU project

Who are the target groups?

Cysec is neither limited to topics like cyber security nor focused on small and medium sized enterprises (SME). The versatile nature of the coaches allows to cover arbitrary topics and is therefore suitable for other parties, such as

  • Private persons
  • Associations
  • Companies
  • Governments
  • Non government organizations

Data security

Cysec does store the answers which were given by the users but of course we are fully compliant to the EUs data security agreement.

Specific coaching

You can install diverse coaches you want to use. A coach is not a survey tool like monkeysurvey but an intelligent digital assistant. Questions can be enriched with text, images and videos to provide more context information.

Coaches react on the users input in a variety of different ways: Based on a users input it might include additional questions on the fly to dig deeper into a subject. Coaches are also able to give recommendations on specific questions to guide a user towards best practices.

Gamification is a well-known driver of motivation, therefore Cysec uses scores, grades and achievements as stimulus to motivate users to keep going. The overall progress for a user is also displayed in the skillboard
